# ComboForge

ComboForge is a mobile-first striking training app for boxing, kickboxing, and
Muay Thai. It combines a round timer, combo callouts, custom combo management,
session history, statistics, speech settings, and local JSON import/export.

The app is guest-first and local-first: core training features work without an
account, and user data is stored locally in the browser.

## Tech Stack

- React
- TypeScript
- Vite
- Vitest

## Scripts

```bash
npm run dev
npm run test:run
npm run lint
npm run format:check
npm run build
```
