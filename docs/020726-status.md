### Currently Solid

Core navigation: Home, Train, Combos, History, Settings.
Round timer: start, pause, end, round/rest transitions.
Combo rotation: current/upcoming combo, discipline-based combo pool.
Starter libraries: Boxing, Kickboxing, Muay Thai.
Custom combo creation + local persistence.
Session history: save/discard summary, list, details modal, delete.
Theme switching and local persistence.
Mobile-first UI direction is largely aligned with the design system.

### Missing For Spec v1

Text-to-speech: no browser speech synthesis yet for combos, interruptions, round transitions.
Countdown: no 3, 2, 1, Fight pre-session countdown.
Difficulty presets: spec wants Easy/Normal/Hard, currently only intensity on combos and fixed comboRotationIntervalMs.
Custom difficulty settings: no UI/model for pace, interruption frequency, offensive/defensive interruptions, combo sources.
Interruptions: no interruption model or training insertion logic yet.
Combo editing/deleting: create exists, but edit/delete custom combos are missing.
Session history fields: currently stores date/duration/rounds/discipline, but not difficulty or combos used.
Statistics: no sessions completed, total training time, total rounds completed summary.
JSON export/import: missing.
Settings: only theme exists; no training defaults, speech settings, import/export, etc.
Combo sources: training currently uses all combos matching discipline; no starter-only/custom-only/selected-combos control.
Probably Later / Spec-Ambitious
Background music: listed as a feature, but not in MVP list. I’d defer.
IndexedDB/Dexie: spec says potential implementation. Current localStorage is fine for now.
Stance: type exists, but no UI or behavior around Orthodox/Southpaw yet.

### Suggested Next Order

Finish custom combo management: edit/delete custom combos.
Track combosUsed in session history.
Add basic stats to History.
Add TTS for combo callouts.
Add difficulty presets by mapping pace to comboRotationIntervalMs.
Biggest gap for “final frontend v1” is probably difficulty + speech + history completeness. Those are the pieces that make it feel like a training app rather than a timer with cards.
